>Author：梅子酒

#### 题目描述: 

对一个框架的反序列化进行利用链挖掘

#### docker-compose.yml

```yaml
version: '3.2'

services:
  mysql:
    image: registry.cn-hangzhou.aliyuncs.com/n1book/web-logic-mysql:latest
    environment:
      -  MYSQL_ROOT_PASSWORD=Cy4Qndv95KKrZCT1
  web:
    image: registry.cn-hangzhou.aliyuncs.com/n1book/web-logic-web:latest
    ports:
      - 80:80
    links:
      - mysql
```

#### 启动方式

`docker-compose up -d`
